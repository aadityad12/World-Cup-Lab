# Public 2026 Match Feature Data Sources

Generated by `ingestion/build_public_match_features.py`.

## Connected public/no-key sources
- Fixtures: `https://raw.githubusercontent.com/openfootball/worldcup.json/master/2026/worldcup.json`. This is a public community dataset, not an official FIFA API.
- Team ratings / Elo ranks / fixtures cross-check: `https://www.eloratings.net/fixtures.tsv`, `https://www.eloratings.net/latest.tsv`, `https://www.eloratings.net/en.teams.tsv`.
- Venues/stadium coordinates: `data/host_venues_2026.csv` static host venue reference.
- Weather: Open-Meteo forecast API at stadium coordinates and kickoff date/hour.

## Important limitations
- Official FIFA fixtures API was not connected; if you have an official schedule feed, use it instead.
- Player injuries, player status, suspensions, and expected lineups were not available from reliable no-key public APIs.
- Attack/defense/midfield/set-piece features are derived proxies from Elo + recent results, not provider-grade event data.
- Travel fatigue includes rough venue-to-venue great-circle distance once a team has already played in the tournament.
- Weather availability depends on Open-Meteo coverage for the requested dates.
